# twitch-eggtimer-bot
Twitch EggTimer bot

## Usage

## Start a timer

Use !eggtimer start {name} {time} to start a timer

Possible time options:
- 1-59s
- 1-59m
- 1-{x}h

### Stop a timer

Use !eggtimer stop {name} to stop a timer

### Get time left

Use !eggtimer time {name} to get the time left
